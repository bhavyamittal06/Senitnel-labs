How it works

Six layers of autonomous reasoning

Senitnel doesn't match rules. It reasons — across accounts, across time, across teams — the way a senior analyst would.

01 — Ingest
Watches your log stream
Reads transactions as they arrive. Never stores raw data. Processes and moves on.
02 — Filter
Ignores 99% of noise
Lightweight Go rules eliminate obvious noise before LLM reasoning begins. Only real signals advance.
03 — Correlate
Connects the dots
6-month memory traces hidden relationships across accounts. Same origin. Same destination. Same timing.
04 — Reason
LLM sees the full picture
Fine-tuned model reasons about the group — not individual events. Identifies FATF typologies and RBI implications.

Architecture

Runs inside your infrastructure.
Nothing leaves.

One Docker container. Zero external API calls in production. Architecturally incapable of sending your data anywhere.

Private deployment
Ships as a Docker container pulled into your environment. You control everything. We never have access to your systems.
Local LLM inference
Fine-tuned model runs inside the container. No Claude API. No OpenAI. No data leaves your network for inference.
Three-layer memory
Hot memory (2 hours), warm memory (30 days), cold baseline (6 months). Catches fast attacks and slow ones alike.
PCI-DSS & RBI compliant by design
Data localisation satisfied. No third-party data processor. Audit trail included. Compliance conversation is simple.
One command installation
docker pull + docker run. No agents on every machine. No SDK integration. No 6-month implementation project.
